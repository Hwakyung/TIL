## 파이썬 기본 문법

### if문(조건문)
```python
if(True):
	print('if문 안 쪽입니다.')
```
```python
if dust > 150 :
    print('매우나쁨')
elif dust > 80  and dust <= 150:
    print('나쁨')
elif 30 < dust <= 80 :
    print('보통')
else :
    print('좋음')
    
```
>dust값에 따른 문자열을 출력하는 조건문이다.
>

### 내장함수 
- `print()`는 파이썬이 가지고 있는 내장함수이다.
- `abs()`는 파이썬이 가지고 있는 절대값을 만들어 주는 내장함수이다.

### 변수 

- 변수는 값을 저장하는 공간이다. 

  > `dust = 60`  dust라는 변수에 60이라는 숫자의 값을 저장한다.
  >
  > `greeting = '안녕하세요'` 등 문자열도 값을 저장할 수 있다.
  >
  > '=='는 비교연산자이며 '='의 값의 할당임으로 주의를 해야한다.

### 리스트 

- 여러 값을 하나의 변수에 저장하기 위한 공간이다.
- 리스트의 값을 호출하기 위해서는 점심메뉴[0]들 대괄호안에 주소 값을 입력해주어야 한다.
- 리스트의 주소값은 0부터 시작함으로 주의해야 한다. 
>`점심메뉴 = ['비빔밥', '육개장','라면']` 하나의 변수 안에 여러 정보가 들어가 있다.

### 딕셔너리 

- key와 value 가 한 쌍으로 연결된 자료형이다.
- 딕셔너리의 값을 호출하기 위해서는 phonebook['삐아짜'] 대괄호 안에 키값을 입력해야 한다.
- 이 때, 키 값이 문자이며 따음표를 붙어 문자열이라는 것을 인식하게 해야한다.
>```python
>phonebook = {
  '삐아짜' : '062-227-6100',
  '김가네' : '062-528-9981',
  '돼지똥꼬' : '062-513-9285'
}
>```
>phonebook이라는 변수 안에 가게 이름이라는 key 값과 전화번호라는 value값이 저장되어 있다.



### while문(반복문)

- 반복적인 코드 작성은 while문을 통해서 쉽게 작업할 수 있다.

  ```python
  while 조건 :
  	수행할 문장1
  	수행할 문장2
  	
  ```

- while문의 조건이 참일 경우, 반복해서 수행되어진다. 이는 무한 루프라는 문제가 발생할 수 있음으로 코드를 벗어날 조건을 항상 설정해주어야 한다.



### for문(반복문)

- 효율성을 얻기 위해서 반복문을 사용한다.
```python
for 변수 in 리스트(또는 튜플, 문자열):
    수행할 문장1
```
#### 예시
  ```python
  for i in range(0,3) :
  	print(i)
  ```

  range()는 범위를 나타내는 함수로 결과값으로 0,1,2가 출력된다.

  

  ```python
  import random
  num = random.sample(range(1,46), 6) 
  
  print(num)
  ```

   > import는 외부의 함수를 가져와서 효율적으로 사용할 수 있게 해준다.
   >
   > range(1,46)은 1부터 45까지의 숫자의 리스트를 만들며 이는 random.sample에 의해 6개의 랜덤한 숫자를 추출하여 num 이라는 변수에 저장하게끔 한다.
