- 서버에 배포를 한다

  - 컴파일 : 사용자가 작성한 코드를 컴퓨터가 이해할 수 있는 언어로 바꿔줌
  - 빌드 : 컴파일된 코드를 실제 수행할 수 있는 상태로 만드는 일
  - 배포 : 빌드가 완성된 실행 가능한 파일을 사용자가 접근 가능할 수 있는 환경에 배치시키는 일
  - 혹은 컴파일을 포함한 war, jar 등의 실행 가능한 파일을 뽑아내기까지의 과정을 빌드라고 함

  빌드 자동화, 배포 자동화 개념
